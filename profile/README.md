# evnchn-AR7778

Projects around the **AR-7778 musical calculator** — a 12-digit desktop calculator that doubles as a piano (notes C4–B6).

| Repository | Description | Language |
|---|---|---|
| [AR7778-digitized-MIDI](https://github.com/evnchn-AR7778/AR7778-digitized-MIDI) | Arduino Nano + solenoids to turn the calculator into a MIDI instrument | C++ |
| [musical-calculator-web](https://github.com/evnchn-AR7778/musical-calculator-web) | Web-based emulator of the calculator's musical functionality | HTML |

See also: [Hackster.io project page](https://www.hackster.io/evnchn/musical-calculator-on-the-web-ar7778-5dc06e)
